Excel File Format

Your Excel file should have the following columns (in any order):

Staff ID: Required (e.g., BBHCF001)
Name / Faculty Name: Required – full name for display (e.g., Mr.Umesh, Prof. K. Umesh Shetty)
Designation: Optional (e.g., Professor & Principal)
Department: Optional (e.g., Administration)
Category: Optional (default: Teaching Faculty)
Email: Optional
Username: Optional (default: Staff ID lowercase)
Password: Optional (default: Staff ID + '123')

Maximum file size: 10MB. First row should contain column headers.
